# 💰 AI-Powered Finance Tracker

A full-stack finance tracking web app with AI-powered insights to help users manage their spending intelligently.

---

## 🚀 Features

* 🔐 User Authentication (JWT)
* 💸 Add Income & Expenses
* 📊 Interactive Charts & Analytics
* 🤖 AI Insights (Spending analysis)
* 📄 Export Reports (PDF)
* 📅 Monthly & Category Breakdown

---

## 🛠️ Tech Stack

### Frontend

* React / Next.js
* Tailwind CSS
* Recharts

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Tools

* JWT Authentication
* OpenAI API

## ⚙️ Installation & Setup

### 1️⃣ Clone the repo

```
git clone https://github.com/your-username/finance-tracker.git
cd finance-tracker
```

### 2️⃣ Backend setup

```
cd backend
npm install
npm run dev
```

### 3️⃣ Frontend setup

```
cd frontend
npm install
npm run dev
```
